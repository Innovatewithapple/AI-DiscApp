# 📱 Cloud IDE (iOS MVP)

This is a **super simple Cloud IDE mobile app** built with **Swift + UIKit**.  
It allows users to chat with an LLM (Gemini via OpenRouter) and preview the generated output (HTML, text, or images).

📷 Demo
👉 Demo Video Link:
https://drive.google.com/file/d/1VccHJJ6AL-W1pjLSC0i_YRVHVS4tm7UQ/view?usp=drive_link

⚡ Note: This is an **iOS-only client app**.  
No custom backend services are used — all requests are made directly from the iOS app to the OpenRouter API.

---

## ✨ Features
- **Chat Interface**: WhatsApp-style bubble UI for user & AI messages.  
- **Text + Image Support**: AI can return text, code snippets, or Base64 images.  
- **Typing Indicator**: Shows "..." bubble while AI is generating a reply.  
- **Live Preview**: HTML responses are rendered inside `WKWebView`.  
- **Async/Await Networking**: Clean and modern Swift concurrency.  

---

## 🛠️ Tech Stack
- **UIKit + AutoLayout** → dynamic chat bubble UI  
- **UITableView** → chat-style message list  
- **WKWebView** → live preview of generated websites  
- **Async/Await + URLSession** → API calls  
- **OpenRouter API** → free backend (using `google/gemini-2.5-flash-image-preview:free`)
